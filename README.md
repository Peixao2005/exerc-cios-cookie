Exercício 2 - Loja virtual

Em uma loja virtual, o ideal é usar sessions para manter o usuário logado, porque é mais seguro. Também dá pra usar sessions no carrinho de compras, já que os produtos ficam guardados enquanto a pessoa navega no site. Já os cookies podem ser usados para salvar preferências, como idioma ou produtos que o usuário já viu. Isso ajuda a melhorar a experiência sem precisar de dados importantes. Então, cada um tem sua função: sessions para segurança e cookies para praticidade.
